Documentation XML, exemple

Cet exemple montre comment documenter du code à l'aide de code XML. Pour plus d'informations, consultez .

Pour générer l'exemple Documentation XML dans Visual Studio

  1. Dans l'Explorateur de solutions, cliquez avec le bouton droit sur le projet, puis cliquez sur Propriétés.

  2. Ouvrez le dossier Propriétés de configuration, puis cliquez sur Générer.

  3. Attribuez à la propriété Fichier de documentation XML la valeur XMLsample.xml.

  4. Dans le menu Générer, cliquez sur Générer. Le fichier de sortie XML se trouve dans le répertoire de débogage.

Pour générer l'exemple Documentation XML à partir de la ligne de commande

  1. Pour générer l'exemple Documentation XML, tapez le texte suivant à l'invite de commandes : 

    csc XMLsample.cs /doc:XMLsample.xml

  2. Pour afficher le code XML généré, exécutez la commande suivante : 

    type XMLsample.xml
